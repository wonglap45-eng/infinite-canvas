# EONS AI Image Studio 用户手册

适用对象：公司内部员工、运营、美工、电商图片制作人员。

本文档说明如何使用 EONS AI Image Studio 进行提示词生成、参考图生图、局部编辑、素材管理和图片下载。

## 1. 登录系统

1. 打开公司提供的 EONS AI Image Studio 网址。
2. 进入登录页后输入管理员分配的用户名和密码。
3. 登录成功后进入工作台。

注意：

- 登录状态会保留 7 天。
- 如果管理员修改了密码，需要重新登录。
- 不要把账号密码发给公司外部人员。

## 2. 数据隔离说明

当前版本主要把画布、素材、生成记录和图片缓存保存在员工自己的浏览器中。

通常情况下：

- 不同电脑使用，不会看到彼此的本地项目和素材。
- 不同浏览器或不同浏览器用户使用，通常也是独立数据。
- 同一台电脑、同一个浏览器用户使用，会看到同一份本地数据。

重要提醒：

- 清理浏览器缓存、IndexedDB 或站点数据，可能导致本地项目和素材丢失。
- 当前版本不是完整的多员工账号系统，暂不支持跨电脑自动同步个人项目。

## 3. 主界面模块

系统主要包含以下功能区：

- 我的画布：适合复杂图片流程、参考图连接、提示词节点、生图节点、局部编辑。
- 生图工作台：适合快速文生图、图生图、批量生成和下载。
- 视频创作台：保留原有视频能力。
- 提示词库：查找和复制现成提示词。
- 我的素材：保存常用图片、提示词和参考素材。
- 错误日志与成本控制：查看生成记录、失败原因和内部估算点数。

## 4. 推荐工作流：亚马逊商品主图

适合制作白底主图、产品场景图、A+ 图和包装优化图。

1. 进入我的画布或生图工作台。
2. 上传产品参考图。
3. 点击“亚马逊工作流”。
4. 选择合适模板：
   - 亚马逊白底主图
   - 产品场景图
   - 卖点强化图
   - A+ 详情质感图
   - 换背景不换产品
   - 包装高级化
   - 多角度变体组图
5. 在提示词末尾补充具体要求。
6. 检查模型、尺寸、数量和质量。
7. 点击生成。
8. 在生成前确认窗口里检查风险提示。
9. 生成后选择满意图片保存或下载。

建议：

- 商品图尽量上传清晰参考图。
- 如果要保留品牌、标签、瓶型、盒型、配色，请在提示词里明确说明。
- 一次生成 1 到 2 张更适合控成本；需要选图时再生成多张。

## 5. 文生图

文生图适合没有参考图、只根据文字生成概念图。

操作步骤：

1. 进入生图工作台或创建一个生图节点。
2. 输入画面描述。
3. 选择图片模型。
4. 设置比例、质量和生成数量。
5. 点击生成。

提示词建议包含：

- 画面主体
- 使用场景
- 构图角度
- 背景
- 光线
- 风格
- 禁止项，例如不要水印、不要额外文字、不要变形

## 6. 图生图

图生图适合基于已有产品图生成新图片。

操作步骤：

1. 上传参考图。
2. 输入修改或生成要求。
3. 使用“亚马逊工作流”模板补充专业约束。
4. 确认参考图数量不是 0。
5. 点击生成。

常见用法：

- 白底主图优化
- 换背景
- 包装质感升级
- 产品角度变化
- 电商详情图
- 生活场景图

## 7. 局部编辑

局部编辑适合只改图片的一小块区域。

操作步骤：

1. 在画布中选中一张图片。
2. 点击“局部编辑”。
3. 用画笔涂抹要修改的区域。
4. 输入修改要求。
5. 点击 AI 修改。

提示词示例：

```text
只修改蒙版区域，其他区域保持不变。把瓶盖改成白色，保持原有光影、材质和瓶身标签不变。
```

注意：

- 蒙版区域越准确，结果越稳定。
- 不要涂太大区域，否则模型可能重绘过多内容。
- 当前模型如果不支持蒙版局部编辑，系统会提示不支持。

## 8. 生成提示词

如果只知道简单需求，可以先生成专业提示词。

操作步骤：

1. 点击底部工具栏的“生成提示词”入口。
2. 输入简单需求。
3. 系统会生成更完整的中文提示词。
4. 把生成的提示词连接到图片生成节点，或复制到生图输入框。

建议：

- 商品图提示词尽量使用中文，方便人工检查。
- 生成提示词后仍要检查是否有品牌、包装、标签、比例、背景和禁止项约束。

## 9. 提示词库

提示词库用于查找灵感和快速复制模板。

操作：

- 按标题搜索。
- 按分类和标签筛选。
- 点击卡片查看详情。
- 复制提示词。
- 加入我的素材。

提示词库的封面只是参考图，不代表最终一定生成同样效果。

## 10. 我的素材

我的素材用于保存常用图片和提示词。

可以保存：

- 参考产品图
- 成品图
- 常用提示词
- 商品图模板

注意：

- 素材主要保存在当前浏览器本地。
- 换电脑后不会自动同步。
- 清理浏览器数据可能导致素材丢失。

## 11. 下载图片

生成完成后，可以：

- 查看大图。
- 下载图片。
- 保存到我的素材。
- 复制提示词再次生成。

建议下载最终确认图，并在公司文件夹或网盘中额外备份。

## 12. 点数说明

系统里的点数是内部估算值，用来提醒成本和避免误消耗，不代表第三方平台真实账单。

估算规则：

```text
估算点数 =
（操作基础值 + 参考图数量 x 0.2 + 长提示词加成）
x 质量系数
x 尺寸系数
x 生成张数
```

基础值：

```text
提示词生成：0.25
文生图：1
图生图：1.35
局部编辑：1.55
每张参考图：+0.2
提示词超过 1000 字：+0.15
```

质量和尺寸：

```text
high：x 1.7
medium：x 1.25
auto/low：x 1

2K 附近尺寸：x 1.35
4K：x 2.4
普通尺寸：x 1
```

真实费用请以第三方平台账单为准。

## 13. 常见问题

### 生成结果很差怎么办？

优先检查：

- 是否上传了参考图。
- 是否用了亚马逊工作流模板。
- 是否明确要求保留品牌、包装、标签位置、主色和比例。
- 是否写了不要水印、不要额外文字、不要扭曲 logo、不要虚构包装。
- 是否一次生成太多导致难以控制。

### 为什么提示没有参考图？

可能原因：

- 图片没有真正连接到生图节点。
- 当前选中的节点不是图片节点。
- 参考图没有被激活。
- 使用了纯文生图入口。

### 为什么别人看不到我的画布？

当前项目数据主要保存在你自己的浏览器本地，不是云端共享项目。

### 换电脑后为什么看不到原来的项目？

当前版本暂不支持个人云端同步。需要后续增加多用户账号和服务端存储。

### 登录不了怎么办？

联系管理员检查 Railway 环境变量：

```env
ADMIN_USERNAME
ADMIN_PASSWORD
```

如果管理员刚修改密码，请刷新页面后重新登录。
